# DFW-app

This is the real DFW platform environment.

DFW-app is the user-facing application that the `dfw-life` hatching system should build over time.

## Product direction
- map-first life operating system
- interactive map as primary interface
- life organized across:
  - logbook
  - future
- action-oriented surface for human life

## Relationship to dfw-life
- `dfw-life` = nervous system / cockpit / builder
- `DFW-app` = the real platform being grown

## Current build principle
Start narrow.
Build real primitives.
Preserve the map-first DFW thesis.

## Current MVP capabilities
- logbook / future map layers
- manual place pinning and planning
- manual run logging
- GPX and TCX run import into the Logbook layer with route rendering
- FIT import is supported
- data persists locally in the browser for the MVP
